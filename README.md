# 메이플 스토리 길드원 변경점 확인기

메이플 스토리 게임에서 길드 관리를 편하게 도와주는 웹사이트

                    
## 구축 환경           
VMWare                   
Ubuntu                    
Docker              
nginx                


## 사용 언어
JavaScript    
html       
css       

                          
## 주요 기능
게임내 길드원의 정보를 처음 접속했을 때 저장 후 사용자가 원하면 처음 저장했던 데이터와 비교해 신규로 가입한 길드원이나 탈퇴한 인원이 있는지 변경점을 확인 합니다.

                          
## -
-로그인       
-길드 검색              
-길드 멤버 저장        
-이후 인원의 변경점 비교 확인 (캐릭터의 닉네임)        
-변경점 확인 후 새로 저장
